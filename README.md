# Jarvis - AI Voice Based Desktop Assistant

Jarvis is a Python-based AI desktop assistant that can perform tasks using voice and text commands. It helps automate daily computer activities such as opening applications, searching the web, telling time, and providing useful information.

## Features

- Voice command recognition
- Text command support
- Open applications automatically
- Web search functionality
- Time and date information
- Simple and user-friendly interface

## Tech Stack

- Python
- SpeechRecognition
- Pyttsx3
- OpenAI API
- OS module
